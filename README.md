# Firecrawl E-commerce Scraper 🛒

A simple Python tool that demonstrates how to use the **Firecrawl SDK** to extract structured product data from e-commerce websites and save it to a CSV file.

## 🚀 Features
- Uses [Firecrawl](https://www.firecrawl.dev/) to handle complex web scraping.
- Extracts structured JSON data.
- Exports results directly to CSV.

## 🛠️ Setup
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Get your API Key from [Firecrawl](https://www.firecrawl.dev/).

## 💻 Usage
Run the script by providing the URL and your API key:
```bash
python scraper.py --url "https://example-ecommerce-store.com" --key "YOUR_API_KEY"
```
